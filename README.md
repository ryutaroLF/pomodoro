
# What Is This ?

1. Run the .py code on your windows machine
2. Counting down will starts and show the remaining time on the task-bar.
3. When the time is over, all display will blackout and the new count down will start for the rest time.

# Install

## Conda Environment Setup

```bash
conda create -n pomodoro python=3.11 -y
conda activate pomodoro
python -m pip install --upgrade pip
pip install pystray pillow screeninfo
```

## Run

I reccomend to put .bat file on your STREAM DECK so that you can easily start the pomodoro timer.

```Bash
Python pomorodo.py
```
